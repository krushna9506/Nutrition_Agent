## ЁЯЪА Quick Start

### 1. Prerequisites

- Python 3.9 or higher
- An IBM Cloud account ([Sign up free](https://cloud.ibm.com/registration))
- An IBM Watsonx.ai project ([Get started](https://dataplatform.cloud.ibm.com/wx/home))

### 2. Get IBM Cloud Credentials

**Step A тАФ Create an API Key:**
1. Go to [IBM Cloud IAM тЖТ API Keys](https://cloud.ibm.com/iam/apikeys)
2. Click **Create an IBM Cloud API key**
3. Name it (e.g., `nutriai-key`) and copy the key immediately

**Step B тАФ Get your Watsonx Project ID:**
1. Open [IBM Watsonx.ai](https://dataplatform.cloud.ibm.com/wx/home)
2. Open your project тЖТ **Manage** tab тЖТ **General**
3. Copy the **Project ID** (UUID format)

### 3. Configure Environment

```bash
# Copy the example env file
cp .env.example .env

# Edit .env and add your credentials
notepad .env          # Windows
nano .env             # Linux/macOS
```

Set these values in `.env`:
```env
IBM_CLOUD_API_KEY=your_actual_api_key
WATSONX_PROJECT_ID=your_actual_project_id
WATSONX_URL=https://us-south.ml.cloud.ibm.com
```

### 4. Install Dependencies

```bash
# Create a virtual environment (recommended)
python -m venv venv

# Activate it
venv\Scripts\activate        # Windows
source venv/bin/activate     # Linux/macOS

# Install packages
pip install -r requirements.txt
```

### 5. Run the Application

```bash
python app.py
```

Open your browser at: **http://localhost:5000**

## ЁЯФз Customizing the Agent (IMPORTANT)

All agent behavior is controlled in **`agent_instructions.py`** тАФ no other files need editing.

### Sections you can customize:

| Section | What to change |
|---|---|
| `PERSONA` | Name, tone, communication style |
| `EXPERTISE` | Add/remove health domains |
| `INDIAN_FOOD_PREFERENCES` | Regional cuisines, festivals, measurements |
| `DIET_SPECIALIZATIONS` | Add new diet types (e.g., Mediterranean, GAPS) |
| `RESPONSE_FORMAT` | How AI structures its responses |
| `SAFETY_RULES` | Disclaimer rules, banned advice |
| `FAMILY_INSTRUCTIONS` | Family plan behavior |
| `CALORIE_FORMULAS` | Override formulas if needed |

**Example тАФ Change the agent name and make it more casual:**
```python
# In agent_instructions.py
PERSONA = """
You are Mitali, a fun and friendly nutrition coach from Mumbai.
You use simple language, mix a little Hindi (like 'bilkul!' and 'arre!'),
and make healthy eating feel exciting and achievable...
"""
```










## тШБя╕П IBM Cloud Code Engine Deployment

```bash
# Install IBM Cloud CLI + Code Engine plugin
ibmcloud login --apikey $IBM_CLOUD_API_KEY
ibmcloud ce project select --name nutriai-project

# Create secrets
ibmcloud ce secret create --name nutriai-secrets \
  --from-literal IBM_CLOUD_API_KEY=$IBM_CLOUD_API_KEY \
  --from-literal WATSONX_PROJECT_ID=$WATSONX_PROJECT_ID

# Deploy
ibmcloud ce app create \
  --name nutriai-agent \
  --image icr.io/your-namespace/nutriai-agent:latest \
  --env-from-secret nutriai-secrets \
  --port 5000
```
